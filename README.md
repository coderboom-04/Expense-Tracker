# 💰 Expense Tracker

A JavaScript-based expense tracking application with visual charts.

## Features
- ✅ Add income and expenses
- 📊 Visual breakdown with Chart.js
- 🔍 Filter by category and type
- 📥 Export to CSV
- 💾 Persistent storage with Local Storage

## Technologies
- JavaScript (ES6+)
- HTML5 & CSS3
- Chart.js
- Local Storage API
